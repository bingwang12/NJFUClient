# NJFUClient
----------------
南林跑操系统电脑端

## 目录结构
```
doc/					文档
src/					代码
	controller/			控制器
		DBConnecter		数据库连接器
		DBController		数据库控制器
		ExcelImporter		表格导入器
		FileSelectControl	文件选择器
		ImportControl		导入控制器
		PayCardControl		刷卡控制器
		RecordSyncer		记录同步器
		UpdateCardContorl	换卡控制器
	main/				主包
		Main			主类
	model/				模型
		Student			学生
		Record			记录
		StudentRecord		总记录
	view/				视图
		MainFrame		主窗口
```
